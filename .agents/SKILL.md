
# SKILL: context-optimization

## Description
This skill optimizes text for Large Language Models (LLMs) by compressing it without sacrificing readability.

## Usage
To use this skill, simply provide the text to be optimized and adjust the compression level as needed.

## Compression Levels
* Low: 30% reduction in file size
* Medium: 50% reduction in file size
* High: 70% reduction in file size

## Example Use Case
```markdown
# Original Text
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

# Optimized Text (Medium Compression)
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
```

## Notes
* This skill uses a combination of techniques to compress text, including:
	+ Removing whitespace and unnecessary characters
	+ Using run-length encoding for repeated sequences
	+ Applying Huffman coding for more efficient compression
